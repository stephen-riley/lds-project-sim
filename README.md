# PBR sim

## Files

- `docs/` - contains original documentation for the exercise
- `stats.xlsx` - modified stats from the original spreadsheet, prepared for use in the simulation
- `sim1.pl` - the Monte Carlo simulator. Note the contents of `stats.xlsx` are copied into this program.
- `pbr_results.xlsx` - an example set of results

## Usage

`perl sim1.pl --battle-iterations 10000 -sim-iterations 250`
